# 智能付款追踪器 (Payment Tracker)

一个专为旅游行业设计的智能付款管理工具，旨在自动化和简化对供应商的付款流程。该项目完全借助 **AI 编程助手 (Cursor)** 高效开发，展示了利用 AI 工具构建完整前端应用的能力。

---

## 🎯 项目背景

在传统旅游业务中，财务人员需要手动跟踪大量订单、计算给不同供应商（如酒店、地接社、导游）的付款日期和金额，过程繁琐且容易出错。本项目旨在解决这一痛点，通过自动化数据导入和任务生成，大幅提升效率和准确性。

## ✨ 核心功能

- **多源数据导入**: 支持从 **Excel (多工作表批量导入)**, **CSV**, 以及 **飞书多维表格** 导入订单、产品和供应商数据。
- **智能付款任务生成**: 根据订单的出发日期和服务的付款条款（如“定金50%+尾款50%”），自动计算并生成待办付款任务。
- **可视化看板管理**: 以看板（Kanban）形式清晰展示所有付款任务的状态（待处理、已处理、已逾期），一目了然。
- **灵活的数据关联**: 智能关联订单、客户、服务和供应商，构建完整的业务数据链路。

## 🤖 AI 驱动的开发流程

本项目是 **AI 驱动开发 (AI-Driven Development)** 的一次成功实践，严格遵循“人机协同”的最佳模式，与贵公司要求的鸿蒙 APP 开发流程高度一致：

1.  **需求定义**: 首先，我负责清晰地定义项目需求、功能模块和业务逻辑。
2.  **AI 编码实现**: 随后，我高效利用 **Cursor (AI 编程工具)**，通过自然语言指令，让 AI 快速生成前端组件、业务逻辑和数据处理脚本的完整代码。
3.  **调试与优化**: 在 AI 生成代码的基础上，我进行快速的功能测试、调试和迭代优化，确保应用的功能稳定和用户体验流畅。这类似于鸿蒙 APP 根据审核反馈进行修改和调试的过程。

这个流程不仅极大地提升了开发效率，也证明了我具备快速学习并高效运用新型 AI 工具解决复杂业务问题的能力。

## 🛠️ 技术栈

- **前端框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **UI 库**: Element Plus
- **样式方案**: Tailwind CSS
- **开发语言**: JavaScript

## 🚀 如何本地运行

```bash

npm install

# 2. 启动开发服务器
npm run dev
```
